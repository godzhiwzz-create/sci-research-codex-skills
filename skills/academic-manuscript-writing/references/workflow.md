# Workflow

Use this workflow when drafting or revising a manuscript.

1. Collect source material.
   - Open manuscript sections, figure assets, tables, experiment summaries, and any README files that describe status.
2. Classify each source.
   - Label each item as final, representative final, verified summary, raw verification source, or unsafe intermediate.
3. Build an evidence map.
   - Record `Claim | Evidence file | Evidence role | Finality | Risk | Notes`.
4. Decide the paper role for each experiment.
   - Use phenomenon-establishing, mechanism-discriminating, reinforcement/robustness, consequence/method implication, or generalization/boundary.
5. Draft after evidence roles are fixed.
   - Write the prose around the evidence, not the other way around.
6. Decide main text vs supplementary.
   - Keep the main text focused on the argument chain.
   - Move repetitive or weakly diagnostic material out of the main line.
7. Re-check claim strength.
   - Keep universal claims out unless the evidence truly supports them.

If a source is mixed-status or flagged by a README, treat it as unsafe until re-verified.
