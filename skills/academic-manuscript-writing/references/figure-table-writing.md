# Figure and Table Writing

## Caption formula
Write captions in the order:
what is measured -> what setting or comparison is shown -> what pattern appears -> why it matters.

## Surrounding prose formula
Use one short paragraph around each figure or table:
- name the comparison
- state the observed pattern
- explain why the pattern supports or weakens the claim

## Rules
- Make one figure or table answer one argumentative question.
- Keep prose focused on interpretation, not just values.
- Avoid repeating every number in the caption and again in the body.
- If a table duplicates the visual message of a figure, compress or move one of them to supplementary material.
- Use strong but calibrated verbs:
  - supported
  - suggests
  - indicates
  - remains insufficient

## Micro-template
```text
What is measured -> What pattern appears -> Why it matters
```

## Bad patterns to avoid
- Number dumping without interpretation
- Captions that only restate the axis labels
- Claims that are broader than the plotted evidence
- Figure prose that does not explain why the result supports the conclusion
