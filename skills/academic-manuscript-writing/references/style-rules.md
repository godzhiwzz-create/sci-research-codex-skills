# Style Rules

## Claim calibration
- Use “supported in this setting” when the evidence is strong but scoped.
- Use “among the tested mechanisms” when comparing hypotheses.
- Use “representative validation” when the result is not meant to be universal.
- Use “suggests” or “indicates” when the evidence is informative but not decisive.
- Use “establishes” or “shows” only when the evidence is direct and stable.

## Strong versus safe verbs
- Strong: establish, show, support, demonstrate, confirm.
- Safe: suggest, indicate, align with, remain consistent with, are insufficient to explain.

## Common failure modes
- Overstating causality from correlation alone.
- Repeating matrix values instead of explaining the pattern.
- Letting supplementary evidence take over the main line.
- Turning a mechanism paper into a method-hype paper.

## Journal English defaults
- Prefer concise, declarative sentences.
- Prefer one argumentative point per paragraph.
- Keep the conclusion narrow.
- Keep the discussion focused on scope, limitations, and implications.
- Use English-first output unless the user explicitly asks for bilingual notes.
