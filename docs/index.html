这套工具解决什么问题

科研项目最容易失控的地方，不是单个实验不会跑，而是方向、文献、实验、证据和写作分散在不同文件里。这里把它拆成一条闭环：先明确问题，再读论文，再设计验证，最后把结果写进可审计的论文证据链。

方向判断问题和根因，避免陷入无意义调参。
文献先建立来源、主题、验证状态和阅读路线。
实验用实验卡、索引和结果路径保存证据。
写作先做 claim-evidence map，再写论文主张。
审计检查证据、文件、结论和归档是否一致。
